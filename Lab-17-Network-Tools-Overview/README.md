# Lab 17: Network Tools Overview

## Objective

Learn basic Linux networking tools.

## Commands Used

sudo ifconfig

ip addr show

ping -c 4 8.8.8.8

nslookup example.com

dig example.com

## Results

- Displayed network interface information using ifconfig.
- Viewed detailed IP configuration using ip addr.
- Tested connectivity to Google's DNS server.
- Retrieved DNS records using nslookup.
- Queried DNS information using dig.
