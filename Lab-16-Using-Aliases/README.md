# Lab 16: Using Aliases

## Objective
Learn how to create, use, and remove temporary aliases in Linux.

## Commands Used

alias ll='ls -l'

alias

ll

unalias ll

ll

## Output

- Created alias named ll.
- Verified alias using alias command.
- Executed ll successfully.
- Removed alias using unalias.
- Verified removal by receiving "command not found".
