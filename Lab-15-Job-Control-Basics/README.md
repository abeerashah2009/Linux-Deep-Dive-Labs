# Lab 15 - Job Control Basics

## Objective
Learn Linux job control using foreground and background processes.

## Commands Used

```bash
sleep 100
Ctrl + Z
jobs
bg
jobs
fg %1
Ctrl + C
```

## Commands Learned

- sleep
- Ctrl + Z
- jobs
- bg
- fg

## Result

Successfully learned how to suspend, resume, and manage foreground and background jobs in Linux.
